## Columns

Websites often use multiple columns. Create a two column layout for your magazine.

--- task ---

Select the `index.html` file and add code to create two columns. 

--- code ---
---
language: html
filename: index.html
line_numbers: true
line_number_start: 7
line_highlights: 9-14
---
<body>
<h1>My magazine</h1>
<div class="column1">
	Column 1
</div>
<div class="column2">
	Column 2
</div>
</body>

--- /code ---
